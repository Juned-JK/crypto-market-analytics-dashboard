# 🚀 Cryptocurrency Market Analysis Dashboard

## 📌 Project Overview

This project is an end-to-end Cryptocurrency Market Analysis Dashboard built using Python, SQL, Tableau, and Power BI.

The goal of this project is to collect real-time cryptocurrency data from the CoinMarketCap API, perform data cleaning and analysis, and create interactive dashboards that provide insights into market capitalization, trading volume, volatility, gainers, losers, and risk distribution across the crypto market.

## 🎯 Objectives

* Collect cryptocurrency market data using API integration
* Perform data cleaning and transformation using Python
* Store and analyze data using SQL
* Identify market trends and opportunities
* Build interactive dashboards in Tableau and Power BI
* Perform risk and volatility analysis

## 🛠️ Tech Stack

### Data Collection

* Python
* CoinMarketCap API
* Pandas

### Data Storage & Analysis

* MySQL
* SQL

### Data Visualization

* Tableau
* Power BI

## 📊 Key Metrics

* Total Coins Tracked
* Total Market Capitalization
* 24H Trading Volume
* CEX Trading Volume
* DEX Trading Volume

---

## 📈 Dashboard Features

### Market Capitalization Analysis

* Top 10 Cryptocurrencies by Market Cap
* Bottom 10 Cryptocurrencies by Market Cap
* Market Share Analysis

### Trading Volume Analysis

* Top 10 Coins by 24H Trading Volume
* Market Cap vs Trading Volume Comparison

### Performance Analysis

* Top 10 24H Gainers
* Top 10 24H Losers

### Risk Analysis

Coins are categorized based on volatility into:

* 🟢 Low Risk
* 🟡 Medium Risk
* 🔴 High Risk

Metrics analyzed:

* Number of Coins
* Average 24H Change
* Total Market Capitalization
* Total Trading Volume

### Opportunity Detection

* High Volume / Low Market Cap Coins
* Low Volume / High Market Cap Coins


## 📂 Project Workflow

1. Extract cryptocurrency data from CoinMarketCap API
2. Clean and transform data using Pandas
3. Store processed data in MySQL
4. Perform SQL-based exploratory analysis
5. Create visualizations using Tableau and Power BI
6. Publish dashboards for interactive exploration
